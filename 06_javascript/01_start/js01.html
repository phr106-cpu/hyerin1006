JavaScript 사용 방법

JavaScript 이야기

  • JavaScript와 Java는 완전히 다른 프로그래밍 언어이며,
  • 각각의 언어는 특정한 목적과 환경에 맞게 설계되었습니다.
  • 두 언어의 이름이 비슷한 것은 당시에 ‘자바’가 인기있는 언어였기 때문에,
  • 좀더 친숙하게 보이도록 하는 마케팅 전략의 일환이었으며,
  • 실제로는 서로 관련이 없습니다.

Inline JavaScript 사용하기

  • HTML 요소 내에 직접 JavaScript를 작성하는 방법입니다.
  • 코드 양을 적게 작성할 수는 있는데, HTML과 JavaScript 코드가 혼재하여 가독성과 유지 보수에 좋지 않습니다.
  • 팝업 창을 열고 닫거나, 알림 메시지를 표시할 때처럼 간단한 명령을 처리할 경우 인라인 스크립트를 자주 사용 합니다.
  • 아래 type이 button인 button 요소를 클릭하게 되면 onclick으로 연결된 alert 메서드가 호출되게 되고, 시스템 대화상자로 "클릭했어요?"라는 문구가 화면에 나타나게 됩니다

Internal JavaScript(내부 스크립트) 사용하기

  • HTML 파일 내부에서 script 요소를 사용해서 JavaScript 코드를 작성하는 방법입니다.
  • head 요소 또는 body 요소 내부에 위치할 수 있지만, HTML DOM Tree가 생성되고 난 후에 적용되게 하기 위해서 주로 body 요소 닫기 직전에 작성하게 됩니다.
  • 이 방법도 유지보수와 가독성에 취약하므로 주의가 필요합니다.

External JavaScript(외부 스크립트 연결) 사용하기

  • JavaScript 코드를 작성하고 HTML 파일에서 이를 로드해서 사용하는데,
  • 가장 권장되는 방법이며, 코드의 재사용성과 유지 보수가 용이합니다.
