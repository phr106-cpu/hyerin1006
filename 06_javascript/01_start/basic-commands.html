JavaScript 시작하기

1. 식(Expression)이란?

식(Expression)은 자바스크립트 코드 중에서 어떤 값을 만들어 내는 코드 조각을 말합니다.

예를 들어 아래 코드는 모두 식입니다:

  • 5 → 숫자 5라는 값
  • "안녕하세요" → 문자열 값을 만들어냄
  • 2 + 3 → 5라는 값을 계산해서 만듦
  • sum * 2 → 변수 sum의 값에 2를 곱한 값

이렇게 식은 값이 만들어지는 코드라고 기억하면 됩니다.

2. 문(Statement)이란?

문(Statement)은 자바스크립트가 실행하는 명령어 단위입니다. 즉 프로그램에게 동작을 지시하는 코드 단위입니다.

문은 식과 다르게, 명령을 수행하지만 반드시 값을 되돌려주지는 않습니다.

예를 들어:

  • let sum = 5; → 변수에 값을 저장하는 문
  • if (sum > 3) { ... } → 조건에 따라 코드를 실행하는 문
  • for (let i=0; i < 3; i++) { ... } → 반복문

모든 문은 하나 이상의 식을 포함할 수 있습니다.

3. 식(Expression)과 문(Statement) 기본 예제

자바스크립트에서는 식과 문을 구분합니다.

  • 식(Expression) : 실행하면 값을 만들어내는 코드 조각입니다.
  • 문(Statement) : 하나의 명령어로, 동작을 지시합니다.

아래 예제를 실행하고 결과를 확인해보세요.

const sum = 3 + 2; // 식: 5라는 값을 생성
const greeting = '안녕하세요'; // 식: 문자열 값
const isAdult = 20 >= 18; // 식: true 값을 생성
if (sum > 5) {
output2 = 'sum 값이 5보다 큽니다.';
} else {
output2 = 'sum 값이 5 이하입니다.';
}

4. 자바스크립트 입출력 명령어 소개

자바스크립트에서 가장 기본적인 입출력 방법은 아래와 같습니다.

  • alert() : 알림 창 출력
  • prompt() : 사용자에게 입력 받기
  • confirm() : 사용자 확인(예/아니오) 받기
  • document.write() : HTML 문서에 직접 출력
  • console.log() : 개발자 도구 콘솔에 출력
